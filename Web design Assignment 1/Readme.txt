README - Massachusetts Innovation Hub Website

=============================================

This readme file provides an overview of the HTML code structure for the Massachusetts Innovation Hub website.

HTML Tags Used:
---------------

1. `<!DOCTYPE html>` - Document type declaration specifying HTML5.
2. `<html>` - Root element.
3. `<head>` - Contains metadata and links to external resources.
    - `<meta charset="UTF-8">` - Character set declaration.
    - `<meta name="viewport" content="width=device-width, initial-scale=1.0">` - Viewport configuration.
    - `<link rel="icon" href="connection-letter-o-vector-logo-260nw-1095943598.webp" type="image/x-icon">` - Favicon link.
    - `<link rel="stylesheet" href="styles.css">` - Link to external CSS file.
    - `<title>` - Page title.

4. `<body>` - Main content of the webpage.
    - `<details open>` - Details element with the "open" attribute.
        - `<summary>` - Summary of the Massachusetts Innovation Hub website.

    - Sections:
        - `<header>` - Website header containing navigation links.
            - `<h1>` - Main heading.
            - `<nav>` - Navigation links.
                - `<ul>` - Unordered list.
                - `<li>` - List item.
                - `<a>` - Anchor (link) element.

        - `<section id="about">` - About Us section.
            - Various text content, multimedia elements (audio, video), and a table.

        - `<section id="whatweoffer">` - What we offer section.
            - Text content, an ordered list, and a form with input fields and checkboxes.

        - `<section id="Community">` - Community section.
            - Text content, an unordered list with testimonials.

        - `<section id="contact">` - Contact Us section.
            - Text content and an address element.

        - `<footer>` - Website footer.
            - Text content.

5. `<img>` - embeds the Image element.
6. `<audio>` - embeds the Audio element.
7. `<video>` - embeds the Video element.
8. `<table>` - used to create Table .
9. `<ul>` - used to create Unordered list .
    - `<li>` - Used to create List item .
10. `<ol>` - Ordered list element.
    - `<li>` - List item element.
11. `<form>` - Form element.
    - Various form controls: `<label>`, `<input>`, `<textarea>`, `<button>`.
